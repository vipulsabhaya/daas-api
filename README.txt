
This folder contains a simple implementation of the DaaS API in Java.  This implementation will be used for demo purposes.

Prerequisites:
  Eclipse IDE (JEE Developers version will be best)
  M2Eclipse plugin - Can be found at update site: http://download.eclipse.org/technology/m2e/releases/
  Maven

Importing:
  Import this project as a Maven Project into eclipse.

Building:
  From command line, run 'mvn package' to build the daas.war deployable archive.
